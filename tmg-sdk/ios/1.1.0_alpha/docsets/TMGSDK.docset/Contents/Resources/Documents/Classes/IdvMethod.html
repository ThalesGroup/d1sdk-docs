  • value

    Only for IdvType.appToApp. It corresponds to the concatenation of the issuer application id and the intent handler. An information that could be displayed to the end user.

    • For IdvType.otpEmail, it is a masked email.
    • For IdvType.appToApp, it is the issuer mobile application name.
    • For IdvType.customerService, it is the number to call like 1-800-555-1212.
