D1PushToECommerce

public class D1PushToECommerce

Main interface of the push provisioning to eCommerce.

Since

3.3.0

Enrol API for Mastercard click to pay push provisioning

  • pushToClick2PayMC(_:consumerInfo:privacyURL:tncURL:completion:)

    Push Digital Card to Mastercard Click to Pay TokenRequestor. When it is successful, the tokenization process is completed. Integrator may call D1Task.digitalCardList(_:completion:) to confirm. Before calling this Click to Pay API, integrator should display the default value of consumerInfo, collect the end user inputs of consumerInfo, show privacy and T&C contents, and ensure that the end user accepts the contents.

    The URL container for the consumer’s consent status to the Terms and Conditions.
