CardState

public enum CardState : String

Card state.

Since

1.0.0
  • active

    Virtual Card is in an Active state. It can be used for operations.

    Declaration

    Swift

    case active = "ACTIVE"
  • inactive

    Virtual Card is in an Inactive state. It cannot be used for operations.

    Declaration

    Swift

    case inactive = "INACTIVE"
  • expired

    Virtual Card has expired. It needs to be renewed.

    Declaration

    Swift

    case expired = "EXPIRED"
  • deleted

    Virtual Card is deleted and cannot be used further.

    Declaration

    Swift

    case deleted = "DELETED"
  • replaced

    Virtual Card has been replaced.

    This value only applies for card issued by D1.

    Declaration

    Swift

    case replaced = "REPLACED"
  • pendingIDV

    Virtual Card is in a Pending state. It is reserved for future usage.

    Declaration

    Swift

    case pendingIDV = "PENDING_IDV"
