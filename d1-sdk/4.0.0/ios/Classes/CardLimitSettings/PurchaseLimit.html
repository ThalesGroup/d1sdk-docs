public class PurchaseLimit

The purchase limit of the card issued by D1.

  • periodType

    The limit type for the period.

    Not available for daily period.

    Declaration

    Swift

    public internal(set) var periodType: PeriodType? { get }
  • fixedPeriodIndex

    The period index for weekly, monthly or yearly fixed period.

    • For weekly fixed periof, it represent the day of week the period start (such as 1 for Monday, 2 for tuesday, and so on.);
    • For monthly fixed period, it represent the day of month the period start;
    • For yearly fixed period, it represent the month of the year the period start.

    Not available for CardService.updateCardLimitSettings(_:settings:completion:).
