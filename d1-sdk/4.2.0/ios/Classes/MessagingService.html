MessagingService

public class MessagingService

The main class for message and notification related APIs for the card issued by D1 and D1ClickToPay. Obtains the instance of the class through D1Task.messagingService().

Since

4.0.0
  • registerMessageNotification(_:)

    To register the notification for card issued by D1 and D1ClickToPay.

    The SDK registers the token to server. Hence, updatePushToken(_:completion:) should be called before calling this API. Otherwise, the SDK returns D1Error.Code.pushTokenNotFound.
