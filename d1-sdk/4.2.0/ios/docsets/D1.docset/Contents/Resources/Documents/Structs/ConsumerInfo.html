    middleName

    The consumer’s middle name. Optional. Maximum length is 35 characters.
    lastName

    The consumer’s last name. Maximum length is 35 characters.
    language

    The consumer’s preferred language in the ll-CC format, where ll is the language code and CC is the country or region code (for example, en-US).
    phoneNumberCountryCode

    The country calling code in ITU-T E.164 format. Must start with a plus sign (+) (for example +65).
    phoneNumber

    The consumer’s phone number without the country code and without a leading zero (for example, 91234567).
    email

    The consumer’s email address in RFC 5322 format (for example myemail@mail.com).
