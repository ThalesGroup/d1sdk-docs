  • enrol(_:consumerInfo:name:billingAddress:completion:)

    Enroll the card in Click to Pay. Once the enrollment is successful, the tokenization process is completed. The enrollment status in the integrator system will be updated via a push notification.

    Before calling this Click to Pay API, the integrator should display the default values of ConsumerInfo and BillingAddress and collect the end user inputs for ConsumerInfo.

    Since

    4.2.0

    public func enrol(_ cardID: String,
                  consumerInfo: ConsumerInfo,
                  name: String?,
                  billingAddress: BillingAddress?,
                  completion: @escaping (Status?, D1Error?) -> Void)

    Parameters

    cardID

    The card identifier.
    consumerInfo

    The ConsumerInfo object is initialized with user inputs for first name, middle name, last name, language, phone number, country code, and email.
    name

    The cardholder name as present on the card. It is applicable only to Visa.
    billingAddress

    The BillingAddress associated with the card enrolled in Click to Pay. Mandatory for Visa.
    completion

    The callback for the result. When the operation is successful, Status (Click to Pay status) is returned; otherwise, D1Error is returned.
  • optOutCard(_:completion:)

    Opt-out the card from Click to Pay. Once the opt-out is successful, the tokenization process is completed. The enrollment status in the integrator system will be updated via a push notification.
