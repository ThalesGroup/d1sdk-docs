  • D1SetRaspDetectionCallback

    Function to set the detection callback. App integrators have the option to decide how to proceed upon detection.

    Warning

    This callback can only be set ONCE per application runtime. Subsequent function calls will be ignored.

    Declaration

    Objective-C

    Swift
