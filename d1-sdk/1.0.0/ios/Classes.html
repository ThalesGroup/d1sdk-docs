  • D1VirtualCard

    The Virtual Card Object For security reason, VirtualCard use NSMutableData as variables type which can be wiped (refer to wipe function). Please use following example to covert the variable from NSMutableData to NSString if need:

    Objective C

    NSString *panString = [[NSString alloc] initWithData:pan encoding:NSUTF8StringEncoding];

    Swift

    let panString = String.init(data: pan, encoding: .utf8)
